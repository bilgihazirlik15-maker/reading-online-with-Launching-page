Help / Support

Quick guide for using myReading Online.

How do I start a reading activity?

Go to the home page, choose your level, or open one of the reading cards. Then click Start or View Lesson.

Where can I see my progress?

Open the Progress page. You can see completed activities, activities in progress, and average score.

How do I join a class?

Use the Join a Class box on the home page. Enter the class code your teacher gives you.

Why can’t I see Teacher or Admin links?

Those links are role-based. Students do not see them. Teachers see Teacher, and admins see both Teacher and Admin.

My activity does not appear in progress.

Refresh the page with Ctrl + F5. If it still does not appear, send feedback with the activity title.

I need more support.

Use the Feedback page to send a bug report or suggestion.

Need help?

You can send a message from the Feedback page.
